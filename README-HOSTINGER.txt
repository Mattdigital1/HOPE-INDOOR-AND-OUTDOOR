HOPE INDOOR & OUTDOOR LLC — HOSTINGER WEBSITE
================================================

WHAT IS INCLUDED
- Responsive multi-page website
- Home, gallery, service areas, contact, privacy, terms, thank-you, and 404 pages
- Eight individual service pages
- Mobile navigation and fixed mobile call/quote buttons
- PHP quote form that emails ephhope@gmail.com
- Basic local SEO metadata, structured data, sitemap, and robots.txt
- Security and HTTPS rules in .htaccess

UPLOAD TO HOSTINGER
1. In Hostinger hPanel, open Websites > Manage > Files > File Manager.
2. Open the public_html folder.
3. Remove Hostinger's default index file if one is present.
4. Upload every item inside this package into public_html. Keep the assets and services folders intact.
5. Confirm the domain has an active SSL certificate. The included .htaccess redirects visitors to HTTPS.
6. Open the website in a private/incognito browser and test navigation, phone links, and both quote forms.

BEFORE PUBLIC LAUNCH
1. Replace every instance of YOUR-DOMAIN.com in:
   - index.html
   - robots.txt
   - sitemap.xml
2. Replace representative stock images with Hope Indoor & Outdoor LLC's real project, crew, truck, and before/after photography. The current image links are visual placeholders and require internet access.
3. Confirm the public service list, service areas, business hours, licensing language, and insurance language with the owner.
4. Confirm the recipient in send-quote.php. It is currently set to ephhope@gmail.com.
5. Test email delivery. Shared-hosting mail can sometimes enter spam. If delivery is unreliable, connect an authenticated SMTP mailbox or route the form into the business CRM.
6. Add verified customer reviews only after the owner supplies them. The website intentionally does not invent testimonials.

FORM NOTES
- The quote form requires name, phone, service, project details, and SMS/contact consent.
- It includes a hidden honeypot field to reduce simple bot spam.
- The server sends a plain-text lead notification and redirects successful submissions to thank-you.html.
- Hostinger must be running PHP with the mail() function enabled.

DESIGN DIRECTION
The site follows the reference website's high-conversion contractor layout—dark header, strong hero, prominent quote form, trust bar, service cards, process, gallery, FAQ, service areas, and final call-to-action—while using an original green/gold visual identity and original Hope Indoor & Outdoor LLC copy.


SEO SETUP BEFORE LAUNCH
-----------------------
1. Replace every instance of YOUR-DOMAIN.com with the final domain name.
2. Add the final website URL to Google Search Console and submit /sitemap.xml.
3. Connect the same domain to the company Google Business Profile.
4. Replace representative stock images with original Hope Indoor & Outdoor project photos and descriptive filenames.
5. Keep the business name, phone number, service areas, and services consistent across the website, Google Business Profile, Facebook, Instagram, and directories.
6. Do not create duplicate city pages with copied text. Add genuinely unique project photos, reviews, and local details before expanding location pages.
